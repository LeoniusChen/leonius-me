Git修改远程仓库地址:  
```shell
git remote set-url origin ***
```

已存在仓库，拉取指定分支到本地:
```shell
git checkout -b dev origin/dev
```
有可能出错，建议先fetch一下:
```shell
git fetch origin
```

拉取所有分支:
```shell
git clone xxx
git branch -r | grep -v '\->' | while read remote; do git branch --track "${remote#origin/}" "$remote"; done
git fetch --all
git pull --all
```
